Software

We prefer to keep our software stack as light as possible, so we will mainly use Java and JavaScript with some tools to support the languages. Use the OS you want.

  • Java
  • Maven
  • Git
  • IntelliJ Community Edition
  • An editor that understands JavaScript such as Atom, Visual Studio Code, or Sublime Text. IntelliJ as mentioned above also works well (and many others).

Services:

  • A GitHub account is preferable

Make sure it works

Please make sure your Java installation works and that your Java class paths, SDK, and dependencies are set-up before the workshop! We only have 2.5 hours of actual work each session, so the less time spent fixing Java and missing/partial installs the better for everyone's sanity.

How to ensure Java is correctly installed

  • Typing java -version in a terminal should not produce an error
  • If you do get an error, ensure your JDK bin folder is on your machine's PATH environment variable. On Windows this is typically C:\Program Files\Java\jdk1.8.0_60\bin or something similar.

How to ensure Maven is correctly installed

  • Typing mvn --version in a terminal should not produce an error
  • If you do get an error, ensure your Maven bin folder is on your machine's PATH environment variable.

If you're using IntelliJ, Maven comes bundled and should just work (inside IntelliJ).

Ensure that you can compile and run a Java project in IntelliJ/your editor
This is important! Seriously!

  • Using your editor - preferably IntelliJ, create a new hello world project and ensure you can build it using Java.
  • Also ensure that you can build and run a single unit test using jUnit. The test doesn't have to do anything, just build and run successfully.
